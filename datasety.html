09 — Architektura techniczna
Nowoczesna infrastruktura danych

Struktura grafowa

Procesy i przepływy modelowane jako graf z dynamicznymi zapytaniami.

Format JSON-LD

Lekki, maszynowo czytelny format z natywnym mechanizmem linkowania.

Interfejs API

REST + GraphQL. Integracja z oprogramowaniem LCA bez manualnego importu.

Aktualizacja kaskadowa

Zmiana wskaźnika źródłowego przelicza automatycznie wszystkie zależne zbiory.

Natywna parametryzacja

Jeden model parametryczny generuje warianty na żądanie.

Pełna trasowalność

Każda wartość posiada metadane: źródło, data, osoba i instytucja.

Eksport: ILCD+EPD, EcoSpold v2, CSV/XLSX. Kompatybilność z OpenLCA, SimaPro, GaBi/Sphera.
